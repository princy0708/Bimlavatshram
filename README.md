**Bimlavatsram** is a full-stack AI-powered e-commerce platform built for a handmade kurta and kurti set business. Customers can browse the catalogue, customize their order using voice commands, preview their design, and submit directly to the admin — who receives an instant notification and calls to confirm.


---

## How It Works

```
Customer Opens App
        ↓
Browse Product Catalogue
        ↓
Customize Design by Voice Command
(color · fabric · embroidery · cut · size)
        ↓
AI Generates Design Preview
        ↓
Customer Finalizes Order
        ↓
Admin Receives Instant Notification
(full customization details + customer contact)
        ↓
Admin Calls Customer to Confirm
```

## Customer Flow

**1. Browse**
Explore the full handmade kurta and kurti set catalogue with photos, fabric details, and available sizes.

**2. Customize by Voice**
Speak your requirements naturally:
> *"I want a dark green kurta with zari border, anarkali cut, floor length, size medium"*

The AI understands and fills in the customization form automatically.

**3. Preview**
See a visual mockup of your customized design before finalizing — no surprises.

**4. Submit Order**
Finalize and submit. The admin receives an instant notification with your full order details.

**5. Confirmation Call**
The admin calls you directly to confirm the order, discuss any final details, and arrange delivery.

## Installation

**1. Clone the repository**
```bash
git clone https://github.com/princy0708/bimlavatsram.git
cd bimlavatsram
```

**2. Install dependencies**
```bash
npm install
```

**3. Set up Supabase**
- Create a project at [supabase.com](https://supabase.com)
- Copy your project URL and anon key
- Create a `.env` file:

```env
REACT_APP_SUPABASE_URL=your_supabase_url
REACT_APP_SUPABASE_ANON_KEY=your_supabase_anon_key
```

**4. Run locally**
```bash
npm start
```

## About the Business

This platform was built to bring a handmade clothing business online — giving customers across India access to custom handmade kurtas without the back-and-forth of WhatsApp orders.
